GitHub Actions – CI/CD automat

Rulează scripturi la fiecare push, pull request etc.


# Exemplu: .github/workflows/ci.yml
name: CI

on: [push, pull_request]

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Verifică HTML
        run: npx html-validate .
      - name: Deploy Pages
        if: github.ref == 'refs/heads/main'
        uses: peaceiris/actions-gh-pages@v3
        with:
          github_token: ${{ secrets.GITHUB_TOKEN }}
          publish_dir: .

Colaborare pe GitHub

  1. Fork → copiezi un repo
  2. Clone → git clone
  3. Branch → git checkout -b fix/bug
  4. Commit → git commit -m "..."
  5. Push → git push origin fix/bug
  6. Pull Request → propui modificarea
  7. Review → comentarii, aprobare
  8. Merge → integrare în main

Securitate pe GitHub

  • Dependabot: actualizează dependințele automat
  • CodeQL: scanare statică pentru vulnerabilități
  • Secret Scanning: detectează chei API expuse
  • 2FA obligatoriu pentru toți utilizatorii (2024)
