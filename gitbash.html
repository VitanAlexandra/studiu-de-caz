Git Bash – Terminalul Esențial pentru Git pe Windows

Git Bash este un mediu de linie de comandă bazat pe Bash (Bourne Again SHell), inclus în instalarea oficială Git pentru Windows. Emulează un terminal Linux, permițând utilizarea comenzilor Git și utilitarelor Unix într-un mod nativ.

Windows
Git Bash – Terminal Unix integrat

Ce este Git Bash?

  • Parte din Git for Windows (MSYS2 + MinGW)
  • Include: bash, git, ssh, curl, nano, grep, etc.
  • Rulează în fereastră Windows, dar cu sintaxă Unix
  • Suport pentru scripturi .sh, alias-uri, variabile de mediu

Instalare Git Bash (pas cu pas)

  1. Descarcă de pe git-scm.com
  2. Rulează instalatorul
  3. Alege opțiunile:
    • Use Git from Git Bash only → recomandat
    • Enable Git Credential Manager
    • Use the nano editor by default
    • Enable file system caching
  4. Finalizează → vei avea Git Bash în meniul Start

Comenzi esențiale în Git Bash


# Navigare
pwd                          # Afișează calea curentă
ls                           # Listează fișiere
ls -la                       # Toate fișierele (inclusiv ascunse)
cd "studiu de caz"           # Intră în folder
cd ..                        # Urcă un nivel
cd ~                         # Mergi în folderul home

# Fișiere și directoare
mkdir nou                    # Creează director
touch index.html             # Creează fișier gol
cp fisier.html backup.html   # Copiază
mv fisier.html alt/          # Mută / redenumește
rm fisier.html               # Șterge
rm -r folder/                # Șterge recursiv

# Vizualizare și editare
cat README.md                # Afișează conținut
nano index.html              # Editează cu nano
code .                       # Deschide în VS Code
start .                      # Deschide folderul în Explorer

# Căutare și filtrare
grep "text" *.html           # Caută în fișiere
find . -name "*.html"        # Găsește fișiere
history                      # Istoric comenzi
Ctrl+R                       # Caută în istoric

Configurare Git în Git Bash


# Identitate
git config --global user.name "Nume Prenume"
git config --global user.email "email@tudomain.com"

# Editor implicit
git config --global core.editor "nano"
git config --global core.editor "code --wait"  # VS Code

# Alias-uri utile
git config --global alias.st status
git config --global alias.co checkout
git config --global alias.br branch
git config --global alias.cm "commit -m"
git config --global alias.lg "log --oneline --graph --all"

# Exemplu: git lg → istoric frumos

Integrare cu alte tool-uri

  • Windows Terminal: setează Git Bash ca profil implicit
